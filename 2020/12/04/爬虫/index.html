python3.6+

[补充：后来在python的bin文件夹下看到了2to3文件]

这一步之后还没有成功的原因是：输入python3默认打开的是3.8.2，要找到python3.8.2的安装路径，但是本机找了半天也没找到。只找到了pyhton3.8.6。

所以需要重新定向，把python3指向这个版本。

vim ~/.bash_profile

末尾添加 ：alias python=”/usr/local/Cellar/python@3.8/3.8.6_1/bin/python3.8”

再执行命令 source ~/.bash_profile

python -V 确定执行成功了

从此以后本机的python版本

python3 —> python 3.8.2

然后到刚才下载的beautifulsoup4的文件夹下，运行 python setup.py install，安装成功，版本4.8.2。
