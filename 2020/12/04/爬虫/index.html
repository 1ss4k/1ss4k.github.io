项目实践1: 耽美书籍

爬取豆瓣上，与耽美有关的书籍，存入 书籍名字 和简介 和 作者 和豆瓣评分和价格 到本地csv文件。

参考 https://turingplanet.org/2020/03/28/%e8%b1%86%e7%93%a3%e7%88%ac%e8%99%ab%e9%a1%b9%e7%9b%ae/#xiang_mu_she_ji

###环境

python3.6+

下载Selenium Web Driver (chrome版本 86.0.4240.198（正式版本） (x86_64))

pip install selenium (版本3.140.0)

参考 https://blog.csdn.net/u011151028/article/details/50338347

pip install beautifulsoup4 (4.8.2)

其中python的安装路径为 /usr/local/Cellar/python@3.8/3.8.6_1

出现错误🙅如下

[补充：后来在python的bin文件夹下看到了2to3文件]

这一步之后还没有成功的原因是：输入python3默认打开的是3.8.2，要找到python3.8.2的安装路径，但是本机找了半天也没找到。只找到了pyhton3.8.6。

所以需要重新定向，把python3指向这个版本。

vim ~/.bash_profile

末尾添加 ：alias python=”/usr/local/Cellar/python@3.8/3.8.6_1/bin/python3.8”

再执行命令 source ~/.bash_profile

python -V 确定执行成功了

从此以后本机的python版本

python2 —> python2.7.17

python —> pyhton 3.8.6

python3 —> python 3.8.2

然后到刚才下载的beautifulsoup4的文件夹下，运行 python setup.py install，安装成功，版本4.8.2。
